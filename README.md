# ♻️ E-Waste Image Classification using EfficientNet

## 📌 Overview

This project implements an **image classification system for electronic waste (E-Waste)** using **EfficientNet**. The model automatically identifies different categories of e-waste, helping improve waste segregation and recycling processes.

---

## 🎯 Objectives

* Classify different types of E-Waste using deep learning
* Reduce manual effort in waste sorting
* Build a scalable AI-based solution for environmental sustainability

---

## 🧠 Model Details

* Architecture: EfficientNet
* Framework: TensorFlow / Keras
* Training Platform: Google Colab (GPU/TPU)

---
